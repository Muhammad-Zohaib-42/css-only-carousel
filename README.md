# Modern CSS Carousel 🎡

A high-performance, smooth carousel built entirely with **CSS-only**. This project explores the latest "bleeding-edge" features of CSS to handle scrolling, navigation, and positioning without a single line of JavaScript.

## 🚀 Features
* **Zero JavaScript:** All logic is handled by the browser's CSS engine.
* **Scroll Markers:** Built-in pagination dots using `::scroll-marker`.
* **Anchor Positioning:** Precise button placement using the new `anchor()` function.
* **Responsive Design:** Works perfectly on desktop and mobile.
* **Dark Mode:** A peaceful, high-end dark theme for night-time browsing.

## 🛠️ Modern CSS Techniques Used

This project was built to experiment with the following specifications:

1.  **Scroll Snap:** Ensuring the cards lock into place.
    * [Learn about Scroll Snap](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_scroll_snap)
2.  **Scroll Markers & Groups:** Creating pagination dots directly in CSS.
    * [Learn about Scroll Markers](https://developer.chrome.com/blog/scroll-marker)
3.  **CSS Anchor Positioning:** Positioning the navigation arrows relative to the carousel edges.
    * [Learn about Anchor Positioning](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_anchor_positioning)
4.  **CSS Variables:** Using `:root` for dynamic theme switching.

## 📂 Project Structure
* `index.html` - The semantic structure of the carousel.
* `style.css` - The layout, animations, and modern CSS features.
